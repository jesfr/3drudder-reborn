# 3dRudder Reborn

Redonne vie à ton **3dRudder** — le contrôleur pédestre de 3dRudder (entreprise en faillite depuis 2023).

Ce logiciel remplace l'application officielle abandonnée et fonctionne **sans installation**, en un seul fichier `.exe`.

---

## Téléchargement

👉 **[Télécharger la dernière version](../../releases/latest)** — un seul fichier `.exe`, pas d'installation.

---

## Fonctionnalités

- **Mode souris** — contrôle le curseur avec les pieds (vitesse, zone morte et précision réglables)
- **Mode clavier** — touches configurables pour les 6 directions
- **Mode manette** — émule une manette Xbox 360 (driver gratuit intégré)
- **Indicateur pied détecté** — alerte si aucun pied n'est posé
- **Étalonnage** — définit la position neutre du périphérique
- **Flash firmware** — met à jour le firmware si nécessaire
- **FR / EN** — interface bilingue
- **Réglages sauvegardés** — tout est mémorisé d'une session à l'autre

---

## Utilisation

1. Brancher le 3dRudder en USB
2. Lancer `3dRudder_Reborn.exe`
3. Choisir le mode (Souris / Clavier / Manette)
4. Cliquer **Activer**

> Le mode Manette nécessite le driver **ViGEmBus** — l'application propose de l'installer automatiquement si absent.

---

## Compatibilité

- Windows 10 / 11 (64 bits)
- 3dRudder NS (firmware REV_1437 ou REV_1442)
- Aucune installation requise, aucun driver propriétaire
